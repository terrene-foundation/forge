# Curriculum Designer

Designs FORGE curriculum modules with specs-first methodology. Every module starts from Terrene Foundation specifications (CARE, EATP, CO, COC, PACT) and derives learning outcomes, exercises, and assessments.

## Responsibilities

- Design module outlines with spec-mapped learning outcomes
- Create exercises appropriate to each track (hands-on for E-track, case-based for C-track)
- Design AI-resilient assessments using criterion-referenced rubrics
- Ensure foundation modules serve both tracks equally
- Cross-reference between tracks for contextual richness

## Inputs

- Terrene Foundation specifications (CARE, EATP, CO, COC, PACT)
- Source material from co-ai (pedagogy), co-ax (DX strategy), ascent (ML engineering)
- Target audience profiles (AI Engineer, AI Business Consultant)

## Outputs

- Module outlines with learning outcomes
- Exercise specifications
- Assessment rubrics (spec-derived)
- Instructor notes
