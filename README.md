# FORGE — Foundation for Orchestration, Reasoning, and Generative Engineering

A Terrene Open Academy programme teaching Cognitive Orchestration (CO) and Claude Operations Codex (COC) skills to AI practitioners.

## Two Tracks, One Foundation

**Foundation (F1-F4)** — shared by both tracks:
- CO Fundamentals (CARE, EATP, Dual Plane, Mirror Thesis)
- COC Architecture (5 layers, agents, skills, rules, hooks)
- Kailash SDK Essentials (workflows, DataFlow, Nexus, Kaizen)
- AI Systems Design (orchestration, MCP, multi-agent)

**AI Engineer (E1-E4):**
- COC Development
- ML Pipeline Engineering
- Agent Engineering
- Production Systems

**AI Business Consultant (C1-C4):**
- DX Strategy Frameworks
- Organizational AI Assessment
- CO for Client Engagement
- Governance & Compliance
